## Setup

```bash
npx designer-mode setup
```

This appends instructions to `.github/copilot-instructions.md`.

For MCP-capable setups, you can also add the MCP server configuration.

## Usage

1. Start the relay server: `npx designer-mode-server &`
2. Open your project with Copilot
3. Click elements in the browser and send requests
4. Ask Copilot to handle designer mode requests
